# GitHub Pagesでapp02を公開する方法

## 手順

### 1. GitHub Pagesを有効にする

1. GitHubリポジトリを開く: https://github.com/shionosatoshi/claude03
2. 「Settings」タブをクリック
3. 左側のメニューから「Pages」をクリック
4. 「Source」で以下を選択：
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. 「Save」をクリック

### 2. app02にアクセスするURL

GitHub Pagesが有効になると、以下のURLでapp02にアクセスできます：

```
https://shionosatoshi.github.io/claude03/app02/
```

※ 通常、デプロイに1-2分かかります

### 3. 友人に共有する方法

友人に以下のURLを送るだけ：

```
https://shionosatoshi.github.io/claude03/app02/
```

**特徴：**
- ✅ 無料
- ✅ HTTPS対応
- ✅ 永続的
- ✅ スマホでも動作
- ✅ PWA対応（ホーム画面に追加可能）

## 📱 スマホでホーム画面に追加する方法

### iOS (iPhone)
1. Safariで上記URLを開く
2. 下部の「共有」ボタンをタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

### Android (Chrome)
1. Chromeで上記URLを開く
2. 右上の「メニュー」（3点リーダー）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

## 🎮 実装された機能

### ✅ タッチ最適化
- チェックボックス: 28px → 32px（タッチしやすい）
- ボタン: 最小44px（Apple推奨サイズ）
- 入力フィールド: 16px（iOS自動ズーム防止）

### ✅ PWA対応

### ✅ スワイプ操作
- **右スワイプ**: ToDoを完了/未完了切り替え

### ✅ レスポンシブデザイン
- スマホ・タブレット・PCに最適化
- タッチデバイスでホバー効果を無効化
- フォントサイズ・パディング調整

## 🔧 トラブルシューティング

### GitHub Pagesが表示されない場合
1. GitHubの「Actions」タブでデプロイ状況を確認
2. 1-2分待ってから再読み込み
3. キャッシュをクリアして再読み込み

### PWAがインストールできない場合
1. HTTPSでアクセスしているか確認
2. Service Workerが正常に動作しているか確認（ブラウザの開発者ツール）
3. 一度キャッシュをクリア

### スワイプが動作しない場合
1. 最新のブラウザを使用しているか確認
2. JavaScriptが有効になっているか確認
3. タッチデバイスで操作しているか確認
