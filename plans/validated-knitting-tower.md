# カレンダーアプリ実装計画

## Context
ユーザーは日記を書けるシンプルなカレンダーアプリを作成したい。技術スタックはHTML + CSS + JavaScript、データはLocalStorageに保存、モダン・ミニマルなデザインで実装する。

## 推奨アプローチ

### ファイル構成
```
claudeapp/
├── index.html      # メインHTML
├── styles.css      # スタイルシート（モダン・ミニマルデザイン）
└── app.js          # アプリケーションロジック
```

### 実装ステップ

#### 1. index.html - 基本構造
- カレンダーグリッド（7列 x 6行）
- 月移動ボタン（前月・次月）
- 選択した日付の日記入力エリア
- 日記の保存・削除ボタン
- 日記がある日付にインジケーター表示

#### 2. styles.css - モダン・ミニマルデザイン
**カラーパレット**:
- 背景: 白/薄いグレー (#f8f9fa)
- メインカラー: 深いブルー (#2563eb)
- テキスト: ダークグレー (#1f2937)
- 強調: サブブルー (#3b82f6)

**デザイン原則**:
- 十分な余白（padding/margin）
- クリーンな線と角丸
- シャドウで奥行き
- レスポンシブデザイン対応
- ホバーエフェクトでフィードバック

**コンポーネント**:
- カレンダーグリッド: CSS Gridで7列レイアウト
- 日付セル: 角丸、ホバー時に背景色変化
- 日記エディタ: クリーンなテキストエリア、フォーカス時にボーダー強調
- ボタン: フラットデザイン、角丸

#### 3. app.js - 機能実装

**主要機能**:

1. **カレンダー表示**
   - 現在の月を計算して表示
   - 曜日ヘッダー（日〜土）
   - 月の最初の日曜日までの空白セル
   - 月末日までの日付を生成
   - 日記がある日付にドットインジケーター

2. **月移動**
   - 前月・次月ボタン
   - カレンダーの再描画
   - 選択中の日付を保持

3. **日付選択**
   - クリックで日付を選択
   - 選択状態のハイライト表示
   - その日の日記データをロード

4. **日記管理**
   - テキストエリアで日記を入力
   - LocalStorageに保存（キー形式: `diary-YYYY-MM-DD`）
   - 保存ボタンで即時保存
   - 削除ボタンでその日の日記を削除

5. **LocalStorage管理**
   - 保存形式: `{"2024-03-28": "今日の日記内容..."}`
   - ロード時に全データを取得
   - カレンダー描画時に日記のある日付を判定

### 技術的詳細

**カレンダー生成ロジック**:
```javascript
// 月の最初の日と最後の日を取得
const firstDay = new Date(year, month, 1);
const lastDay = new Date(year, month + 1, 0);
const startDayOfWeek = firstDay.getDay(); // 0=日曜日
const daysInMonth = lastDay.getDate();
```

**LocalStorageキー構造**:
- エントリごと: `diary-YYYY-MM-DD`
- 全データ取得時にプレフィックスでフィルタ

**イベント処理**:
- デリゲートパターンでカレンダーグリッドのクリックを処理
- 入力時に自動保存オプションも検討

## 検証方法

1. ブラウザで `index.html` を開く
2. カレンダーが正しく表示されることを確認
3. 前月・次月ボタンで移動できることを確認
4. 日付をクリックして日記を入力
5. 保存後、ブラウザをリロードしてもデータが保持されることを確認
6. 日記がある日付にインジケーターが表示されることを確認
7. 日記を削除できることを確認
8. 異なる月に移動して戻った時にデータが保持されることを確認
9. LocalStorageのデータ構造を確認（DevTools）
