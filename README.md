# claudeapp

Claude Code アプリケーション

## app01 - カレンダー日記アプリ

日記を書けるシンプルなカレンダーアプリです。

### 機能
- 📅 月間カレンダー表示
- ✏️ 日記の作成・編集・削除
- 💾 LocalStorageによる自動保存
- 💭 毎日違う名言を表示

### 使い方
1. [`app01/index.html`](app01/index.html) をブラウザで開きます
2. カレンダーから日付を選択します
3. 日記を書いて「保存」ボタンをクリックします
4. データはブラウザのLocalStorageに保存されます

### 技術スタック
- HTML5
- CSS3（モダン・ミニマルデザイン）
- Vanilla JavaScript

## app02 - RPG風ToDoリスト

ゲーム要素を取り入れたToDoリストアプリです。ToDoを完了してボスを倒そう！

### 機能
- ⚔️ ToDo完了でボスにダメージ（1Todo=25ダメージ）
- 🐉 1日4Todo完了でボス撃破（100HP/100ダメージ）
- 👹 様々なボスキャラクター（絵文字）
- 🔄 毎日0時に自動リセット + 手動リセット
- 💾 LocalStorageによるデータ保存
- 📊 今日の進捗と撃破数の表示

### 使い方
1. [`app02/index.html`](app02/index.html) をブラウザで開きます
2. ToDoを入力して「追加」ボタンをクリックします
3. ToDoのチェックボックスをクリックして完了します
4. ボスのHPが減り、4つ完了するとボス撃破！
5. データはブラウザのLocalStorageに保存されます

### 技術スタック
- HTML5
- CSS3（RPG風デザイン、アニメーション）
- Vanilla JavaScript（クラスベース設計）
