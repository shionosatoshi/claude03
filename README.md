# claudeapp

Claude Code アプリケーション

## app01 - カレンダー日記アプリ

日記を書けるシンプルなカレンダーアプリです。

### 機能
- 📅 月間カレンダー表示
- ✏️ 日記の作成・編集・削除
- 💾 LocalStorageによる自動保存
- 💭 毎日違う名言を表示

### 使い方
1. [`app01/index.html`](app01/index.html) をブラウザで開きます
2. カレンダーから日付を選択します
3. 日記を書いて「保存」ボタンをクリックします
4. データはブラウザのLocalStorageに保存されます

### 技術スタック
- HTML5
- CSS3（モダン・ミニマルデザイン）
- Vanilla JavaScript
