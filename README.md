Claude Code アプリケーション

## app01 - カレンダー日記アプリ

日記を書けるシンプルなカレンダーアプリです。

### 機能
- 📅 月間カレンダー表示
- ✏️ 日記の作成・編集・削除
- 💾 LocalStorageによる自動保存
- 💭 毎日違う名言を表示

### 使い方
1. [`app01/index.html`](app01/index.html) をブラウザで開きます
2. カレンダーから日付を選択します
3. 日記を書いて「保存」ボタンをクリックします
4. データはブラウザのLocalStorageに保存されます

### 技術スタック
- HTML5
- CSS3（モダン・ミニマルデザイン）
- Vanilla JavaScript

## app02 - RPG風ToDoリスト 🎮

ゲーム要素を取り入れたToDoリストアプリです。ToDoを完了してボスを倒そう！

### 🌟 オンラインで使う（推奨）

スマホ・PCからブラウザでアクセス：

**📱 [https://shionosatoshi.github.io/claude03/app02/](https://shionosatoshi.github.io/claude03/app02/)**

- ✅ スマホ最適化（タッチ操作、スワイプ対応）
- ✅ PWA対応（ホーム画面に追加可能）
- ✅ オフラインでも動作
- ✅ 友人と共有可能

### 機能
- ⚔️ ToDo完了でボスにダメージ（1Todo=25ダメージ）
- 🐉 1日4Todo完了でボス撃破（100HP/100ダメージ）
- 👹 様々なボスキャラクター（絵文字）
- 📱 **スマホ最適化**：タッチしやすいUI、スワイプ操作
- 📲 **PWA対応**：ホーム画面に追加してアプリのように使える
- 👈 **スワイプ操作**：右スワイプで完了、左スワイプで削除
- 🔄 毎日0時に自動リセット + 手動リセット
- 💾 LocalStorageによるデータ保存
- 📊 今日の進捗と撃破数の表示

### 使い方

**ブラウザで使う場合：**
1. [`app02/index.html`](app02/index.html) をブラウザで開く
2. ToDoを入力して「追加」ボタンをクリック
3. ToDoのチェックボックスをクリックして完了
4. ボスのHPが減り、4つ完了するとボス撃破！

**スマホの場合：**
1. 右スワイプでToDo完了
2. 左スワイプでToDo削除
3. ホーム画面に追加でオフラインでも使用可能

### 技術スタック
- HTML5
- CSS3（RPG風デザイン、アニメーション、レスポンシブ）
- Vanilla JavaScript（クラスベース設計）
- Service Worker（PWA/オフライン対応）
- Manifest.json（ホーム画面追加対応）

## app04 - Weather Forecast 天気予報アプリ ☀️

3日間の天気予報と服装アドバイスを画像付きで分かりやすく表示するアプリです。

### 機能
- 🌤️ **現在地の天気表示**: Geolocation APIで現在地を自動取得
- 🌍 **都市名検索**: 世界中の都市の天気を検索可能
- 📅 **3日予報**: 3日分の天気予報をカード形式で表示
- 👔 **服装アドバイス**: 気温と天候に応じた服装推奨
- 👨👩 **男女切り替え**: ボタン1つで男性用・女性用の服装アドバイスを切り替え
- 🔄 **自動キャッシュ**: 30分間天気データをキャッシュ
- 📱 **PWA対応**: スマートフォンにインストール可能
- 💾 **オフライン対応**: Service Workerでオフライン時も動作

### 使い方

**ブラウザで使う場合：**
1. [`app04/index.html`](app04/index.html) をブラウザで開く
2. 初回起動時にAPIキー設定モーダルが表示されます
3. [WeatherAPI.com](https://www.weatherapi.com/) で無料アカウントを作成し、APIキーを取得
4. APIキーを入力して「保存」
5. 📍ボタンで現在地の天気、または検索バーに都市名を入力

**スマホの場合：**
1. ホーム画面に追加でオフラインでも使用可能
2. 👨👩 ボタンで男女の服装アドバイスを切り替え

### 技術スタック
- HTML5
- CSS3（天気テーマデザイン、アニメーション、レスポンシブ）
- Vanilla JavaScript（クラスベース設計）
- **WeatherAPI.com**（無料枠：月1,000,000回）
- Geolocation API
- Service Worker（PWA/オフライン対応）
- Manifest.json（ホーム画面追加対応）

### 📖 詳しい使い方

[app04/README.md](app04/README.md) を参照してください。

## app06 - ポジティブチャットボット 😊

なんでも肯定的・ポジティブに答えてくれるチャットボットアプリです。

### 機能
- 😊 **ポジティブ返答**: どんな話題も肯定的に返答
- 💭 **感情検知**: 悲しみ、不安、疲労、喜びなどを検知して適切に対応
- 🌈 **多彩な返答パターン**: 各カテゴリで複数のバリエーション
- 💬 **リアルタイムチャット**: スムーズな会話体験
- 🎨 **美しいUI**: グラデーションデザイン

### 使い方
1. [`app06/index.html`](app06/index.html) をブラウザで開く
2. メッセージを入力して送信
3. ポジティブな返答を楽しもう！

### 返答例
- 悲しいこと → 「大丈夫ですよ！あなたは強いんです！💪」
- 不安なこと → 「不安な気持ち、分かります。でも大丈夫！あなたならできる！💫」
- 疲れたこと → 「お疲れ様です！今日も頑張りましたね！😊」
- その他 → 「なるほどね！あなたの視点、素敵ですね！😊」

### 技術スタック
- HTML5
- CSS3（グラデーションデザイン、アニメーション、レスポンシブ）
- Vanilla JavaScript
