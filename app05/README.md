# 💰 割り勘計算機

金額を人数で割る簡易計算ツール。均等割りとランダム割りに対応しています。

## 🌟 機能

### 均等割り
合計金額を人数で均等に割り勘します。端数が出る場合は、最初の人に加算されます。

**例**: 10,000円を5人で均等割り
- 各2,000円

**例**: 1,000円を3人で均等割り
- 334円、333円、333円（端数処理）

### ランダム割り
合計金額が同じになるように、各人にランダムに金額を割り当てます。全員が最低1円以上を受け取ります。

**例**: 10,000円を5人でランダム割り
- 2,500円、1,800円、2,200円、1,500円、2,000円（合計10,000円）

### その他の機能
- **統計情報**: 最大、最小、平均を表示
- **履歴機能**: 過去5回分の計算履歴を保存
- **レスポンシブデザイン**: スマホでも快適に操作
- **PWA対応**: ホーム画面に追加してオフラインで使用可能

## 🚀 使い方

1. **金額を入力**: 合計金額を入力（1円〜1,000,000円）
2. **人数を入力**: 人数を入力（2人〜100人）
3. **割り方を選択**: 均等割りまたはランダム割りを選択
4. **計算**: 「計算する」ボタンをクリック
5. **結果確認**: 各人の金額と統計情報を確認

## 🙏 使い道の例

- **飲み会の割り勘**: 会費を均等に分担
- **ランチ代の分担**: ランダムに金額を割り当ててゲーム性を追加
- **イベント費用の分担**: 複数人で費用を分担する際に使用
- **確シミュレーション**: 確率に基づいた金額配分をシミュレーション

## 💡 ランダム割りのアルゴリズム

ランダム割りでは、以下の手順で金額を割り当てます：

1. 全員に最低1円を割り当てる
2. 残りの金額をランダムに分配
3. 最後の人に残りを全て割り当てる（合計が必ず一致）
4. 配列をシャッフルしてランダム性を保証

これにより、合計金額が必ず一致し、全員が1円以上を受け取ることが保証されます。

## 📦 インストール方法

このアプリはPWAに対応しているため、スマートフォンのホーム画面に追加できます。

### iOSの場合
1. Safariでこのページを開く
2. 共有ボタンをタップ
3. 「ホーム画面に追加」を選択

### Androidの場合
1. Chromeでこのページを開く
2. メニューをタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択
